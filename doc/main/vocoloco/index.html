This is VocoLoco!

An Android Messaging Developed through Qt Quick for Software Design @ UIC

DESCRIPTION

VocoLoco was conceptualized to be a group messaging application that would allow users to record and send audio files as well as text messages to other users. The other innovative component to the application is that a user is able to see who was the last to send a message through an intuitive home screen which shows all the users in a conversation and orders them based on who last posted a message.

UNDERSTANDING THE QML

Unfortuately there does not seem to be any automated documentation solution for QML yet. I did however try to conform to some consistent, in-line, documentation and comment styles. In each QML file, the properties are defined at the top and are typically the first thing in the file. Then the visual markup is the body of the file with comments throughout and finally, at the bottom of each file are the functions that pretain to that file with comments above them describing their use and purpose.

SHORTCOMMINGS

With significant effort, we were not able to implement the storing of recorded audio files on the android phone. It required the use of the JNI and instantiating a MediaRecorder in the native code on the android device from within Qt's framework. This gave us very little debugging information and ultimately stopped us from reaching the project goal of sending voice messages across a group message application. We were able to implement the playing of audio files however. In this case the audio files were given in xml as a url, downloaded to the phone and played in the JNI with a MediaPlayer object. Even though this last piece didn't fall into place, we feel that we still have a competent application and a fulfilling project. Created, by hand the components needed to send and receive messages from one android device to another in real time.
