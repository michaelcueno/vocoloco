LOCOCLIENT

Vocoloco Android Client

Big Picture

QML Application Viewer

The heart and soul of the Android client. Most of the client is written in QML. You initialize the application viewer by passing in a main.qml file and include the directory containing all other qml files and the engine builds the app

HttpManager

Backend for the client written in c++.

QML is a fantastic language, but it doesn't do everything we needed it it do. For example, there are many things involving custom networking that we needed to accomplish to implement the messaging client. Luckiliy, QML is extendable through c++
