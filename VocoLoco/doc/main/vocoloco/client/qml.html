Extensions

QML is a fantastic language, but it doesn't do everything we needed it it do. For example, there are many things invovling custom networking that we needed to accomplish to implement the messaging client. Luckiliy, QML is extendable through c++

    HttpManager network;     // My class
    QmlApplicationViewer viewer;   // Qt's Qml App Viewer

    // make network avaiable to QML
    viewer.rootContext()->setContextProperty("network", &network);
