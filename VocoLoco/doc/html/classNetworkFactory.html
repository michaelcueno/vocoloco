Detailed Description

This class is fed to the QML engine so that it uses this class on each new server request.

This class inherits from QDeclarativeNetworkAccessManagerFactory, and adds some neccessary functionality. First we needed to ingore SSL errors because necessitas doesn't handle them correctly. Also the constructor creates and uses our CookieJar implementation so that the user can stay logged in. This is also necessary for requests to function properly (the QML engine must send the cookie with each request).
